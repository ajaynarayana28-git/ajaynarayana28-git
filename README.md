## 📁 Featured Projects

### 🔹 [AWS Cloud Infrastructure Monitoring and Automation](https://github.com/ajaynarayana28-git/aws-cloud-infrastructure-monitoring-and-automation)

> A complete AWS monitoring and automation solution using CloudWatch, SNS, Lambda, EventBridge, CloudTrail, and VPC Flow Logs.

**What it does:**
- ✅ Monitors EC2 CPU utilization with CloudWatch Alarms
- ✅ Sends email alerts via Amazon SNS
- ✅ Automates EC2 operations using AWS Lambda + EventBridge
- ✅ Audits AWS API activity with CloudTrail → S3
- ✅ Captures network traffic via VPC Flow Logs
